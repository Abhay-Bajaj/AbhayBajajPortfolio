# Personal Portfolio

A responsive, single-page personal portfolio website built with **React** leveraging **JavaScript, HTML, CSS, and SCSS** for modern UI and theming. The site showcases my background, skills, projects, and contact information in a clean, modern layout with light and dark mode support. This project focuses on **frontend architecture, component-driven design, and polished UI/UX**.

---

## Live Demo

**https://abhay-bajaj.github.io/AbhayBajajPortfolio/**

---

## Features

- **Responsive Design**
  - Optimized for desktop, tablet, and mobile devices
  - Flexible layouts using modern CSS and media queries

- **Light / Dark Mode**
  - Theme toggle with consistent styling across all sections
  - Dark mode enabled by default for better visual comfort

- **Interactive Sections**
  - About Me and Skills overview
  - Work experience, education, and achievements
  - Project and open-source highlights
  - Contact section with social links

- **Animations & Visuals**
  - Smooth scroll and fade-in animations
  - Lottie animations for a more engaging experience

- **Sticky Navigation**
  - Header remains accessible while scrolling
  - Smooth in-page navigation between sections

- **Custom Styling**
  - SCSS-based global theming
  - Reusable design tokens for colors, typography, and spacing
